تفسير الآية رقم 60 من سورة البقرة

استسقى, أي: طلب لهم ماء يشربون منه. { فَقُلْنَا اضْرِبْ بِعَصَاكَ الْحَجَرَ } إما حجر مخصوص معلوم عنده, وإما اسم جنس، { فَانْفَجَرَتْ مِنْهُ اثْنَتَا عَشْرَةَ عَيْنًا } وقبائل بني إسرائيل اثنتا عشرة قبيلة، { قَدْ عَلِمَ كُلُّ أُنَاسٍ } منهم { مَشْرَبَهُمْ } أي: محلهم الذي يشربون عليه من هذه الأعين, فلا يزاحم بعضهم بعضا, بل يشربونه متهنئين لا متكدرين, ولهذا قال: { كُلُوا وَاشْرَبُوا مِنْ رِزْقِ اللَّهِ } أي: الذي آتاكم من غير سعي ولا تعب، { وَلَا تَعْثَوْا فِي الْأَرْضِ } أي: تخربوا على وجه الإفساد.
