تفسير الآية رقم 36 من سورة الرعد

يقول تعالى: { وَالَّذِينَ آتَيْنَاهُمُ الْكِتَابَ } أي: مننا عليهم به وبمعرفته، { يَفْرَحُونَ بِمَا أُنْزِلَ إِلَيْكَ } فيؤمنون به ويصدقونه، ويفرحون بموافقة الكتب بعضها لبعض، وتصديق بعضها بعضا وهذه حال من آمن من أهل الكتابين، { وَمِنَ الْأَحْزَابِ مَنْ يُنْكِرُ بَعْضَهُ } أي: ومن طوائف الكفار المنحرفين عن الحق، من ينكر بعض هذا القرآن ولا يصدقه.
{ فمن اهتدى فلنفسه ومن ضل فإنما يضل عليها } إنما أنت يا محمد منذر تدعوا إلى الله، { قُلْ إِنَّمَا أُمِرْتُ أَنْ أَعْبُدَ اللَّهَ وَلَا أُشْرِكَ بِهِ } أي: بإخلاص الدين لله وحده، { إِلَيْهِ أَدْعُو وَإِلَيْهِ مَآبِ } أي: مرجعي الذي أرجع به إليه فيجازيني بما قمت به من الدعوة إلى دينه والقيام بما أمرت به.
