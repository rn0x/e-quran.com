تفسير الآية رقم 48 من سورة الشورى

{ فَإِنْ أَعْرَضُوا } عما جئتهم به بعد البيان التام { فَمَا أَرْسَلْنَاكَ عَلَيْهِمْ حَفِيظًا } تحفظ أعمالهم وتسأل عنها، { إِنْ عَلَيْكَ إِلَّا الْبَلَاغُ } فإذا أديت ما عليك، فقد وجب أجرك على اللّه، سواء استجابوا أم أعرضوا، وحسابهم على اللّه الذي يحفظ عليهم صغير أعمالهم وكبيرها، وظاهرها وباطنها.
ثم ذكر تعالى حالة الإنسان، وأنه إذا أذاقه الله رحمة، من صحة بدن، ورزق رغد، وجاه ونحوه { فَرِحَ بِهَا } أي: فرح فرحا مقصورا عليها، لا يتعداها، ويلزم من ذلك طمأنينته بها، وإعراضه عن المنعم.
{ وَإِنْ تُصِبْهُمْ سَيِّئَةٌ } أي: مرض أو فقر، أو نحوهما { بِمَا قَدَّمَتْ أَيْدِيهِمْ فَإِنَّ الْإِنْسَانَ كَفُورٌ } أي: طبيعته كفران النعمة السابقة، والتسخط لما أصابه من السيئة.
