تفسير الآية رقم 38 من سورة محمد

أنكم { تُدْعَوْنَ لِتُنْفِقُوا فِي سَبِيلِ اللَّهِ } على هذا الوجه، الذي فيه مصلحتكم الدينية والدنيوية.
{ فَمِنْكُمْ مَنْ يَبْخَلُ } أي: فكيف لو سألكم، وطلب منكم أموالكم في غير أمر ترونه مصلحة عاجلة؟ أليس من باب أولى وأحرى امتناعكم من ذلك.
ثم قال: { وَمَنْ يَبْخَلْ فَإِنَّمَا يَبْخَلُ عَنْ نَفْسِهِ } لأنه حرم نفسه ثواب الله تعالى، وفاته خير كثير، ولن يضر الله بترك الإنفاق شيئا.
فإن الله هو { الْغَنِيُّ وَأَنْتُمُ الْفُقَرَاءُ } تحتاجون إليه في جميع أوقاتكم، لجميع أموركم.
{ وَإِنْ تَتَوَلَّوْا } عن الإيمان بالله، وامتثال ما يأمركم به { يَسْتَبْدِلْ قَوْمًا غَيْرَكُمْ ثُمَّ لَا يَكُونُوا أَمْثَالَكُمْ } في التولي، بل يطيعون الله ورسوله، ويحبون الله ورسوله، كما قال تعالى: { يَا أَيُّهَا الَّذِينَ آمَنُوا مَنْ يَرْتَدَّ مِنْكُمْ عَنْ دِينِهِ فَسَوْفَ يَأْتِي اللَّهُ بِقَوْمٍ يُحِبُّهُمْ وَيُحِبُّونَهُ }
تم تفسير سورة القتال، والحمد لله رب العالمين.
