تفسير الآية رقم 23 من سورة إبراهيم

ولما ذكر عقاب الظالمين ذكر ثواب الطائعين فقال: { وَأُدْخِلَ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ } أي: قاموا بالدين، قولا، وعملا، واعتقادا { جَنَّاتٍ تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ } فيها من اللذات والشهوات ما لا عين رأت، ولا أذن سمعت، ولا خطر على قلب بشر، { خَالِدِينَ فِيهَا بِإِذْنِ رَبِّهِمْ } أي: لا بحولهم وقوتهم بل بحول الله وقوته { تَحِيَّتُهُمْ فِيهَا سَلَامٌ } أي: يحيي بعضهم بعضا بالسلام والتحية والكلام الطيب.
