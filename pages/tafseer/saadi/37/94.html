تفسير الآية رقم 94 من سورة الصافات

{ فَأَقْبَلُوا إِلَيْهِ يَزِفُّونَ } أي: يسرعون ويهرعون، أي: يريدون أن يوقعوا به، بعدما بحثوا وقالوا: { مَنْ فَعَلَ هَذَا بِآلِهَتِنَا إِنَّهُ لَمِنَ الظَّالِمِينَ }
وقيل لهم { سَمِعْنَا فَتًى يَذْكُرُهُمْ يُقَالُ لَهُ إِبْرَاهِيمُ } يقول: { تَاللَّهِ لَأَكِيدَنَّ أَصْنَامَكُمْ بَعْدَ أَنْ تُوَلُّوا مُدْبِرِينَ } فوبخوه ولاموه، فقال: { بَلْ فَعَلَهُ كَبِيرُهُمْ هَذَا فَسْئَلُوهُمْ إِنْ كَانُوا يَنطِقُونَ فَرَجَعُوا إِلَى أَنفُسِهِمْ فَقَالُوا إِنَّكُمْ أَنْتُمْ الظَّالِمُونَ ثُمَّ نُكِسُوا عَلَى رُءُوسِهِمْ لَقَدْ عَلِمْتَ مَا هَؤُلَاءِ يَنطِقُونَ قَالَ أَفَتَعْبُدُونَ مِنْ دُونِ اللَّهِ مَا لَا يَنفَعُكُمْ شَيْئًا وَلَا يَضُرُّكُمْ } الآية.
