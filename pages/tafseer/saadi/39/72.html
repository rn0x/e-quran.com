تفسير الآية رقم 72 من سورة الزمر

فـ { قِيلَ } لهم على وجه الإهانة والإذلال: { ادْخُلُوا أَبْوَابَ جَهَنَّمَ } كل طائفة تدخل من الباب الذي يناسبها ويوافق عملها. { خَالِدِينَ فِيهَا } أبدا، لا يظعنون عنها، ولا يفتر عنهم العذاب ساعة ولا ينظرون. { فَبِئْسَ مَثْوَى الْمُتَكَبِّرِينَ } أي: بئس المقر، النار مقرهم، وذلك لأنهم تكبروا على الحق، فجازاهم اللّه من جنس عملهم، بالإهانة والذل، والخزي.
