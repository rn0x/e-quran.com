تفسير الآية رقم 50 من سورة المؤمنون

أي: وامتننا على عيسى ابن مريم، وجعلناه وأمه من آيات الله العجيبة، حيث حملته وولدته من غير أب، وتكلم في المهد صبيا، وأجرى الله على يديه من الآيات ما أجرى، { وَآوَيْنَاهُمَا إِلَى رَبْوَةٍ } أي: مكان مرتفع، وهذا -والله أعلم- وقت وضعها، { ذَاتِ قَرَارٍ } أي: مستقر وراحة { وَمَعِينٍ } أي: ماء جار، بدليل قوله: { قَدْ جَعَلَ رَبُّكِ تَحْتَكِ } أي: تحت المكان الذي أنت فيه، لارتفاعه، { سَرِيًّا } أي: نهرا وهو المعين { وَهُزِّي إِلَيْكِ بِجِذْعِ النَّخْلَةِ تُسَاقِطْ عَلَيْكِ رُطَبًا جَنِيًّا* فَكُلِي وَاشْرَبِي وَقَرِّي عَيْنًا }
