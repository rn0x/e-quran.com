تفسير الآية رقم 16 من سورة الأحقاف

{ أُولَئِكَ } الذين ذكرت أوصافهم { الَّذِينَ نَتَقَبَّلُ عَنْهُمْ أَحْسَنَ مَا عَمِلُوا } وهو الطاعات لأنهم يعملون أيضا غيرها. { وَنَتَجاوَزُ عَنْ سَيِّئَاتِهِمْ } فِي جملة { أَصْحَابُ الْجَنَّةِ } فحصل لهم الخير والمحبوب وزال عنهم الشر والمكروه.
{ وَعْدَ الصِّدْقِ الَّذِي كَانُوا يُوعَدُونَ } أي: هذا الوعد الذي وعدناهم هو وعد صادق من أصدق القائلين الذي لا يخلف الميعاد.
