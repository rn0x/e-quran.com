تفسير الآية رقم 15 من سورة النساء

أي: النساء { اللاتي يَأْتِينَ الْفَاحِشَةَ } أي: الزنا، ووصفها بالفاحشة لشناعتها وقبحها. { فَاسْتَشْهِدُوا عَلَيْهِنَّ أَرْبَعَةً مِنْكُمْ } أي: من رجالكم المؤمنين العدول. { فَإِنْ شَهِدُوا فَأَمْسِكُوهُنَّ فِي الْبُيُوتِ } أي: احبسوهن عن الخروج الموجب للريبة. وأيضا فإن الحبس من جملة العقوبات { حَتَّى يَتَوَفَّاهُنَّ الْمَوْتُ } أي: هذا منتهى الحبس. { أَوْ يَجْعَلَ اللَّهُ لَهُنَّ سَبِيلًا } أي: طريقا غير الحبس في البيوت، وهذه الآية ليست منسوخة، وإنما هي مغياة إلى ذلك الوقت، فكان الأمر في أول الإسلام كذلك حتى جعل الله لهن سبيلا، وهو رجم المحصن وجلد غير المحصن.
