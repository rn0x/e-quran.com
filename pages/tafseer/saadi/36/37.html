تفسير الآية رقم 37 من سورة يس

أي: { وَآيَةٌ لَهُمُ } على نفوذ مشيئة اللّه، وكمال قدرته، وإحيائه الموتى بعد موتهم. { اللَّيْلُ نَسْلَخُ مِنْهُ النَّهَارَ } أي: نزيل الضياء العظيم الذي طبق الأرض، فنبدله بالظلمة، ونحلها محله { فَإِذَا هُمْ مُظْلِمُونَ } وكذلك نزيل هذه الظلمة، التي عمتهم وشملتهم، فتطلع الشمس، فتضيء الأقطار، وينتشر الخلق لمعاشهم ومصالحهم،
