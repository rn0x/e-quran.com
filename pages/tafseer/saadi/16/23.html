تفسير الآية رقم 23 من سورة النحل

{ لَا جَرَمَ } أي: حقا لا بد { أَنَّ اللَّهَ يَعْلَمُ مَا يُسِرُّونَ وَمَا يُعْلِنُونَ } من الأعمال القبيحة { إِنَّهُ لَا يُحِبُّ الْمُسْتَكْبِرِينَ } بل يبغضهم أشد البغض، وسيجازيهم من جنس عملهم { إن الذين يستكبرون عن عبادتي سيدخلون جهنم داخرين }
