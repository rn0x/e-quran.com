تفسير الآية رقم 20 من سورة هود

{ أُولَئِكَ لَمْ يَكُونُوا مُعْجِزِينَ فِي الْأَرْضِ } أي: ليسوا فائتين الله، لأنهم تحت قبضته وفي سلطانه.
{ وَمَا كَانَ لَهُمْ مِنْ دُونِ اللَّهِ مِنْ أَوْلِيَاءَ } فيدفعون عنهم المكروه، أو يحصلون لهم ما ينفعهم، بل تقطعت بهم الأسباب.
{ يُضَاعَفُ لَهُمُ الْعَذَابُ } أي: يغلظ ويزداد، لأنهم ضلوا بأنفسهم وأضلوا غيرهم.
{ مَا كَانُوا يَسْتَطِيعُونَ السَّمْعَ } أي: من بغضهم للحق ونفورهم عنه، ما كانوا يستطيعون أن يسمعوا آيات الله سماعا ينتفعون به { فَمَا لَهُمْ عَنِ التَّذْكِرَةِ مُعْرِضِينَ * كَأَنَّهُمْ حُمُرٌ مُسْتَنْفِرَةٌ * فَرَّتْ مِنْ قَسْوَرَةٍ } { وَمَا كَانُوا يُبْصِرُونَ } أي: ينظرون نظر عبرة وتفكر، فيما ينفعهم، وإنما هم كالصم البكم الذين لا يعقلون.
