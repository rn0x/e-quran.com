تفسير الآية رقم 19 من سورة الفرقان

فلما تبرؤوا منهم قال الله توبيخا وتقريعا للعابدين { فَقَدْ كَذَّبُوكُمْ بِمَا تَقُولُونَ } إنهم أمروكم بعبادتهم ورضوا فعلكم، وأنهم شفعاء لكم عند ربكم، كذبوكم في ذلك الزعم وصاروا من أكبر أعدائكم فحق عليكم العذاب، { فَمَا تَسْتَطِيعُونَ صَرْفًا } للعذاب عنكم بفعلكم أو بفداء أو غير ذلك، { وَلَا نَصْرًا } لعجزكم وعدم ناصركم. هذا حكم الضالين المقلدين الجاهلين كما رأيت أسوأ حكم، وأشر مصير.
وأما المعاند منهم الذي عرف الحق وصدف عنه فقال في حقه: { وَمَنْ يَظْلِمْ مِنْكُمْ } بترك الحق ظلما وعنادا { نُذِقْهُ عَذَابًا كَبِيرًا } لا يقادر قدره ولا يبلغ أمره.
