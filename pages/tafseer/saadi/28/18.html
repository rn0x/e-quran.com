تفسير الآية رقم 18 من سورة القصص

{ فـ } لما جرى منه قتل الذي هو من عدوه { أَصْبَحَ فِي الْمَدِينَةِ خَائِفًا يَتَرَقَّبُ } هل يشعر به آل فرعون، أم لا؟ وإنما خاف، لأنه قد علم، أنه لا يتجرأ أحد على مثل هذه الحال سوى موسى من بني إسرائيل.
فبينما هو على تلك الحال { فَإِذَا الَّذِي اسْتَنْصَرَهُ بِالْأَمْسِ } على عدوه { يَسْتَصْرِخُهُ } على قبطي آخر. { قَالَ لَهُ مُوسَى } موبخا له على حاله { إِنَّكَ لَغَوِيٌّ مُبِينٌ } أي: بين الغواية، ظاهر الجراءة.
