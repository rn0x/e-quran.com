تفسير الآية رقم 27 من سورة القصص

{ قَالَ } صاحب مدين لموسى { إِنِّي أُرِيدُ أَنْ أُنْكِحَكَ إِحْدَى ابْنَتَيَّ هَاتَيْنِ عَلَى أَنْ تَأْجُرَنِي } أي تصير أجيرا عندي { ثَمَانِيَ حِجَجٍ } أي: ثماني سنين. { فَإِنْ أَتْمَمْتَ عَشْرًا فَمِنْ عِنْدِكَ } تبرع منك، لا شيء واجب عليك. { وَمَا أُرِيدُ أَنْ أَشُقَّ عَلَيْكَ } فأحتم عشر السنين، أو ما أريد أن أستأجرك لأكلفك أعمالا شاقة، وإنما استأجرك لعمل سهل يسير لا مشقة فيه { سَتَجِدُنِي إِنْ شَاءَ اللَّهُ مِنَ الصَّالِحِينَ } فرغبه في سهولة العمل، وفي حسن المعاملة، وهذا يدل على أن الرجل الصالح، ينبغي له أن يحسن خلقه مهما أمكنه، وأن الذي يطلب منه، أبلغ من غيره.
