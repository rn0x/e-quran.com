تفسير الآية رقم 92 من سورة طه

تفسير الآيتين 92 و93 :ـ
فأقبل موسى على أخيه لائما له، وقال: { يَا هَارُونُ مَا مَنَعَكَ إِذْ رَأَيْتَهُمْ ضَلُّوا* أَلَّا تَتَّبِعَنِ } فتخبرني لأبادر للرجوع إليهم؟ { أَفَعَصَيْتَ أَمْرِي } في قولي { اخْلُفْنِي فِي قَوْمِي وَأَصْلِحْ وَلَا تَتَّبِعْ سَبِيلَ الْمُفْسِدِينَ }
