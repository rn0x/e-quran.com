تفسير الآية رقم 46 من سورة الأنعام

يخبر تعالى، أنه كما أنه هو المتفرد بخلق الأشياء وتدبيرها، فإنه المنفرد بالوحدانية والإلهية فقال: { قُلْ أَرَأَيْتُمْ إِنْ أَخَذَ اللَّهُ سَمْعَكُمْ وَأَبْصَارَكُمْ وَخَتَمَ عَلَى قُلُوبِكُمْ } فبقيتم بلا سمع ولا بصر ولا عقل { مَنْ إِلَهٌ غَيْرُ اللَّهِ يَأْتِيكُمْ بِهِ } فإذا لم يكن غير الله يأتي بذلك، فلم عبدتم معه من لا قدرة له على شيء إلا إذا شاءه الله. وهذا من أدلة التوحيد وبطلان الشرك، ولهذا قال: { انْظُرْ كَيْفَ نُصَرِّفُ الْآيَاتِ } أي: ننوعها، ونأتي بها في كل فن، ولتنير الحق، وتتبين سبيل المجرمين. { ثُمَّ هُمْ } مع هذا البيان التام { يَصْدِفُونَ } عن آيات الله، ويعرضون عنها.
