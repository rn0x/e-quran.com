تفسير الآية رقم 140 من سورة الأنعام

ثم بين خسرانهم وسفاهة عقولهم فقال: { قَدْ خَسِرَ الَّذِينَ قَتَلُوا أَوْلَادَهُمْ سَفَهًا بِغَيْرِ عِلْمٍ } أي: خسروا دينهم وأولادهم وعقولهم، وصار وصْفُهم -بعد العقول الرزينة- السفه المردي، والضلال. { وَحَرَّمُوا مَا رَزَقَهُمُ اللَّهُ } أي: ما جعله رحمة لهم، وساقه رزقا لهم. فردوا كرامة ربهم، ولم يكتفوا بذلك، بل وصفوها بأنها حرام، وهي من أَحَلِّ الحلال. وكل هذا { افْتِرَاءً عَلَى اللَّهِ } أي: كذبا يكذب به كل معاند كَفَّار. { قَدْ ضَلُّوا وَمَا كَانُوا مُهْتَدِينَ } أي: قد ضلوا ضلالا بعيدا، ولم يكونوا مهتدين في شيء من أمورهم.
