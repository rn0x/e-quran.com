تفسير الآية رقم 33 من سورة الحج

{ لَكُمْ فِيهَا } أي: [في] الهدايا { مَنَافِعُ إِلَى أَجَلٍ مُسَمًّى } هذا في الهدايا المسوقة، من البدن ونحوها، ينتفع بها أربابها، بالركوب، والحلب ونحو ذلك، مما لا يضرها { إِلَى أَجَلٍ مُسَمًّى } مقدر، موقت وهو ذبحها إذا وصلت مَحِلُّهَا وهو الْبَيْتِ الْعَتِيقِ، أي: الحرم كله " منى " وغيرها، فإذا ذبحت، أكلوا منها وأهدوا، وأطعموا البائس الفقير.
