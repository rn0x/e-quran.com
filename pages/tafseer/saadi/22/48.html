تفسير الآية رقم 48 من سورة الحج

{ وَكَأَيِّنْ مِنْ قَرْيَةٍ أَمْلَيْتُ لَهَا } أي: أمهلتها مدة طويلة { وَهِيَ ظَالِمَةٌ } أي: مع ظلمهم، فلم يكن مبادرتهم بالظلم، موجبا لمبادرتنا بالعقوبة، { ثُمَّ أَخَذْتُهَا } بالعذاب { وَإِلَيَّ الْمَصِيرُ } أي: مع عذابها في الدنيا، سترجع إلى الله، فيعذبها بذنوبها، فليحذر هؤلاء الظالمون من حلول عقاب الله، ولا يغتروا بالإمهال.
