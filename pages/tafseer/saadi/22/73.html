تفسير الآية رقم 73 من سورة الحج

هذا مثل ضربه الله لقبح عبادة الأوثان، وبيان نقصان عقول من عبدها، وضعف الجميع، فقال: { يَا أَيُّهَا النَّاسُ } هذا خطاب للمؤمنين والكفار، المؤمنون يزدادون علما وبصيرة، والكافرون تقوم عليهم الحجة، { ضُرِبَ مَثَلٌ فَاسْتَمِعُوا لَهُ } أي: ألقوا إليه أسماعكم، وتفهموا ما احتوى عليه، ولا يصادف منكم قلوبا لاهية، وأسماعا معرضة، بل ألقوا إليه القلوب والأسماع، وهو هذا: { إِنَّ الَّذِينَ تَدْعُونَ مِنْ دُونِ اللَّهِ } شمل كل ما يدعى من دون الله، { لَنْ يَخْلُقُوا ذُبَابًا } الذي هو من أحقر المخلوقات وأخسها، فليس في قدرتهم خلق هذا المخلوق الضعيف، فما فوقه من باب أولى، { وَلَوِ اجْتَمَعُوا لَهُ } بل أبلغ من ذلك لو { يَسْلُبْهُمُ الذُّبَابُ شَيْئًا لَا يَسْتَنْقِذُوهُ مِنْهُ } وهذا غاية ما يصير من العجز.{ ضَعُفَ الطَّالِبُ } الذي هو المعبود من دون الله { وَالْمَطْلُوبُ } الذي هو الذباب، فكل منهما ضعيف، وأضعف منهما، من يتعلق بهذا الضعيف، وينزله منزلة رب العالمين.
