تفسير الآية رقم 45 من سورة الحج

{ فَكَأَيِّنْ مِنْ قَرْيَةٍ } أي: وكم من قرية { أَهْلَكْنَاهَا } بالعذاب الشديد، والخزي الدنيوي، { وَهِيَ ظَالِمَةٌ } بكفرها بالله وتكذيبها لرسله، لم يكن عقوبتنا لها ظلما منا، { فَهِيَ خَاوِيَةٌ عَلَى عُرُوشِهَا } أي: فديارهم متهدمة، قصورها، وجدرانها، قد سقطت عروشها، فأصبحت خرابا بعد أن كانت عامرة، وموحشة بعد أن كانت آهلة بأهلها آنسة، { وَبِئْرٍ مُعَطَّلَةٍ وَقَصْرٍ مَشِيدٍ } أي: وكم من بئر، قد كان يزدحم عليه الخلق، لشربهم، وشرب مواشيهم، ففقد أهله، وعدم منه الوارد والصادر، وكم من قصر، تعب عليه أهله، فشيدوه، ورفعوه، وحصنوه، وزخرفوه، فحين جاءهم أمر الله، لم يغن عنهم شيئا، وأصبح خاليا من أهله، قد صاروا عبرة لمن اعتبر، ومثالا لمن فكر ونظر.
