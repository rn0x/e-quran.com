تفسير الآية رقم 51 من سورة فصلت

{ وَإِذَا أَنْعَمْنَا عَلَى الْإِنْسَانِ } بصحة، أو رزق، أو غيرهما { أَعْرَضَ } عن ربه وعن شكره { وَنَأَى } ترفع { بِجَانِبِهِ } عجبا وتكبرًا. { وَإِنْ مَسَّهُ الشَّرُّ } أي: المرض، أو الفقر، أو غيرهما { فَذُو دُعَاءٍ عَرِيضٍ } أي: كثير جدًا، لعدم صبره، فلا صبر في الضراء، ولا شكر في الرخاء، إلا من هداه الله ومنَّ عليه.
