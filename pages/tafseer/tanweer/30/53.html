تفسير الآية رقم 53 من سورة الروم

وَمَا أَنْتَ بِهَادِي الْعُمْيِ عَنْ ضَلَالَتِهِمْ إِنْ تُسْمِعُ إِلَّا مَنْ يُؤْمِنُ بِآَيَاتِنَا فَهُمْ مُسْلِمُونَ (53)
وقرأ الجمهور : { ولا تسمع الصمّ } بتاء فوقية مضمومة وكسر ميم { تُسمِع } ونصب { الصمَّ } ، على أنه خطاب للنبيء صلى الله عليه وسلم وقرأه ابن كثير { ولا يَسمع الصمُّ } بتحتية مفتوحة وبفتح ميم { يسْمَع } ورفع { الصمّ } على الفاعلية ل { يَسمع } . وقرأ الجمهور { بهادي } بموحدة وبألف بعد الهاء وبإضافة { هادي } إلى { العُمْي } ، وقرأه حمزة وحْده { تهدي } بمثناة فوقية وبدون ألف بعد الهاء على الخطاب وبنصب { العُمْي } على المفعولية .
