تفسير الآية رقم 56 من سورة النمل

فَمَا كَانَ جَوَابَ قَوْمِهِ إِلَّا أَنْ قَالُوا أَخْرِجُوا آَلَ لُوطٍ مِنْ قَرْيَتِكُمْ إِنَّهُمْ أُنَاسٌ يَتَطَهَّرُونَ (56) تقدم نظير هاته الآية في سورة الأعراف ( 82 ) ، وخالفتها هذه بوقوع العطف بالفاء في قوله { فما كان جواب قومه } دون الواو ، وبقوله { أخرجوا ءال لوط } عوض { أخرجوهم } [ الأعراف : 82 ] وبقوله { قدرناها } عوض { كانت } [ الأعراف : 83 ] ، وبقوله { فساء مطر المنذرين } عوض { فانظر كيف كان عاقبة المجرمين } [ الأعراف : 84 ] .
