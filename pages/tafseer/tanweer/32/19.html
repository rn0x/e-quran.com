تفسير الآية رقم 19 من سورة السجدة

أَمَّا الَّذِينَ آَمَنُوا وَعَمِلُوا الصَّالِحَاتِ فَلَهُمْ جَنَّاتُ الْمَأْوَى نُزُلًا بِمَا كَانُوا يَعْمَلُونَ (19) ، فجملةُ { فلهم جنات المأوى } إلى آخرها مؤكدة لمضمون جملة { فلا تعلم نفس ما أخْفِي لهم } [ السجدة : 17 ] إلى آخرها .
وجملة { فمأواهم النار } إلى آخرها مؤكدة لمضمون جملة { فذوقوا بما نسيتم لقاء يومكم هذا إلى بما كنتم تعملون } [ السجدة : 14 ] .
و { مَن } الموصولة في الموضعين عامة بقرينة التفصيل بالجمع في قوله { أمَّا الذينَ آمنوا } الخ .
